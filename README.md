# LXS Tools

Free online utility tools — Password Generator, IMEI Generator, Phone Number Generator, UUID Generator. All tools run 100% client-side in your browser.

Built with Vue 3 + Vite + TypeScript + TailwindCSS.

## Features

- **Password Generator** — Strong, secure, random, memorable passwords + PIN codes
- **IMEI Generator & Validator** — Generate valid IMEI numbers with Luhn checksum
- **Phone Number Generator** — Random phone numbers for 12+ countries
- **UUID Generator** — UUID v4 and v7 with multiple formats

## Development

```bash
npm run dev     # Start dev server
npm run build   # Build for production
npm run preview # Preview production build
```
